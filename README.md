# LeetCode-C-Practice
This repository is dedicated to practicing LeetCode problems and common interview questions using C.

🎯 Goals
Interview Preparation: Strengthening problem-solving skills for technical interviews.

C vs. C++ Comparison: Exploring the syntax and performance differences between the two languages.

Deep Dive into Memory & Pointers: Focusing on low-level concepts such as:

Manual memory management (malloc/free vs. new/delete).

Pointer manipulation and Linked List implementations.

Dynamic data structures (e.g., Hash Tables in C).

🛠️ Environment
IDE: Visual Studio Code

Compiler: MinGW-w64 (GCC/G++)

Build System: Task-based compilation in VS Code
